# itsmeallen Portfolio - Blending Technical Excellence with Ghibli-Inspired Magic

I created a portfolio that combines modern web technologies with Studio Ghibli's enchanting aesthetic. This project showcases both technical skills and creative vision in a cohesive user experience.

## Key Highlights:

- **Interactive Design**: Dynamic animations using Framer Motion create depth and storytelling
- **Light/Dark Themes**: Custom theme system with smooth transitions between modes
- **Secure Admin**: Innovative access methods (keyboard shortcut/four-finger touch)
- **Complete CMS**: Built with Convex backend for content management without code changes
- **Responsive Experience**: Optimized for all devices with tailored interactions

## Tech Stack:
Next.js 13+ | React | TypeScript | Tailwind CSS | Framer Motion | Convex | Vercel

## Creative Elements:

- **Animated Backgrounds**: Day/night environments that change with theme
- **Paper-fold Effects**: Custom CSS creates realistic folded corners on cards
- **Magic Circles**: Interactive rotating elements with particle effects
- **Easter Eggs**: Hidden interactions including Totoro character with special animations

## Challenges Solved:

- **Performance**: Balanced rich visuals with optimized load times
- **Animation**: Created smooth 60fps animations across devices
- **Accessibility**: Built with semantic HTML, ARIA attributes, and keyboard navigation

## Business Value:

This portfolio demonstrates my approach to digital experiences—blending artistic vision with technical execution. It creates a memorable presentation that differentiates my work through thoughtful interactions, cohesive storytelling, and meticulous attention to detail.

The project showcases not just completed work, but how I approach problems, combining creative thinking with technical problem-solving to create experiences that resonate with users.
